Una tantum
==========

* In una directory qualsiasi o globale (-g) *

npm install --save-dev create-react-app

* Creazione app (installa react)  
create-react-app supercalc

* Dalla app react
npm install --save @material-ui/core
npm install --save @material-ui/icons

* npm start  
Parte in test

* npm run build  
Crea l'app
